# ТЗ: Этап 1.2 — Контент «Ритейл Логистик»

## 1. Цель

Заменить контент Kumho Tires на контент первого заказчика — «Ритейл Логистик» (РЛ). РЛ — импортёр и дистрибьютор алкогольной продукции. PHP-ядро не трогаем — работа исключительно с данными, шаблонами, стилями и статикой.

**Результат этапа:** публичная часть сайта РЛ работает, Kumho-контент полностью удалён, визуальный стиль соответствует бренду РЛ, демо-инструменты доступны без авторизации.

### 3.2 Страница «Демо-инструменты» — детализация

Новая страница, не имеющая аналога в Kumho. Показывает AI-возможности платформы без авторизации.

**Секции:**
1. **Promo** — заголовок «AI-инструменты», подзаголовок
2. **Инструмент 1: Удаление фона** — загрузка изображения → API Photoroom → результат. Уже работает: `POST /api/photoroom/remove-background`
3. **Инструмент 2: Извлечение данных с этикетки** — загрузка фото бутылки → (будущий API) → таблица с данными. На этом этапе: заглушка «Скоро» или демо с предзаполненными данными

**Требует:**
- Новый JSON: `data/json/ru/pages/demo-tools.json`
- Новый SEO: `data/json/ru/seo/demo-tools.json`
- Новая секция или content-container с формой загрузки
